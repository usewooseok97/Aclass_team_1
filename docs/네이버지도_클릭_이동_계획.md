# 축제 위치 클릭 시 네이버 지도 이동 기능

## 요청 사항

`FestivalLocationMap.tsx`에서 "위치: {selectedFestival.PLACE}" 텍스트를 클릭하면 네이버 지도 앱/웹으로 이동하여 해당 장소가 검색되도록 구현

---

## 구현 방안

### 1. naverMap.ts에 URL 생성 함수 추가

```typescript
/**
 * 네이버 지도 검색 URL 생성
 * @param place 장소명
 * @param guName 구 이름 (선택)
 * @param mapx 경도 (선택)
 * @param mapy 위도 (선택)
 */
export function getNaverMapSearchUrl(
  place: string,
  guName?: string,
  mapx?: string,
  mapy?: string
): string {
  const query = guName ? `서울 ${guName} ${place}` : place;
  const encodedQuery = encodeURIComponent(query);

  // 좌표가 있으면 좌표 기반 URL
  if (mapx && mapy) {
    const lat = parseInt(mapy) / 10000000;
    const lng = parseInt(mapx) / 10000000;
    return `https://map.naver.com/v5/search/${encodedQuery}?c=${lng},${lat},15,0,0,0,dh`;
  }

  // 좌표가 없으면 검색 URL
  return `https://map.naver.com/v5/search/${encodedQuery}`;
}
```

### 2. FestivalLocationMap.tsx 수정

```tsx
import { ExternalLink } from "lucide-react";
import { getNaverMapSearchUrl } from "@utils/naverMap";

// 위치 텍스트 부분을 클릭 가능한 링크로 변경
{selectedFestival && (
  <a
    href={getNaverMapSearchUrl(
      selectedFestival.PLACE,
      selectedFestival.GUNAME,
      selectedFestival.mapx,
      selectedFestival.mapy
    )}
    target="_blank"
    rel="noopener noreferrer"
    className="text-sm flex items-center gap-1 hover:underline cursor-pointer"
    style={{ color: "var(--btn-primary)" }}
  >
    <ExternalLink className="w-4 h-4" />
    위치: {selectedFestival.PLACE}
  </a>
)}
```

## 동작 방식

1. 사용자가 "위치: 코엑스 A홀, C홀" 텍스트 클릭
2. 새 탭에서 네이버 지도 열림
3. URL 예시: `https://map.naver.com/v5/search/서울%20강남구%20코엑스%20A홀,%20C홀?c=127.0589,37.5126,15,0,0,0,dh`

---

## 검증 방법

1. 앱 실행 후 축제 상세 페이지 이동
2. "위치: {장소명}" 클릭
3. 새 탭에서 네이버 지도가 열리고 해당 장소가 검색되는지 확인
