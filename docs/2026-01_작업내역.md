## 주요 구현 기능 요약

### 1. 시간대별 테마 시스템
- 아침(morning): 주황색 계열
- 낮(day): 파란색 계열
- 석양(sunset): 보라색 계열
- 밤(night): 짙은 보라 계열

### 2. 전역 상태 관리
- FestivalContext로 축제 데이터 관리
- viewMode: 'map' | 'list' | 'detail'
- selectedDistrict, selectedFestival 상태

### 3. 레이아웃 구조
- LeftSectionContainer (40%) + RightSectionContainer (40%)
- 조건부 콘텐츠 렌더링
- 반응형 디자인

### 4. 컴포넌트 아키텍처
```
src/
├── atoms/          # 작은 UI 컴포넌트
├── components/     # 중형 컴포넌트
├── containers/     # 컨테이너/로직 컴포넌트
├── pages/          # 페이지 컴포넌트
├── contexts/       # React Context
├── hooks/          # 커스텀 훅
├── constants/      # 상수
├── types/          # TypeScript 타입
├── utils/          # 유틸리티 함수
└── assets/         # 정적 자산
```
